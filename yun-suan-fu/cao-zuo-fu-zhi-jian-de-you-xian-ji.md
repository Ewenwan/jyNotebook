表格列出了从最高到最低优先级的所有运算符：

| 运算符 | 描述 |
| :--- | :--- |
| \*\* | 指数 \(最高优先级\) |
| ~ + - | 按位翻转, 一元加号和减号 \(最后两个的方法名为 +@ 和 -@\) |
| \* / % // | 乘，除，取模和取整除 |
| + - | 加法减法 |
| &gt;&gt;&lt;&lt; | 右移，左移运算符 |
| & | 位 'AND' |
| ^ \| | 位运算符 |
| &lt;= &lt;&gt;&gt;= | 比较运算符 |
| &lt;&gt; == != | 等于运算符 |
| = %= /= //= -= += \*= \*\*= | 赋值运算符 |
| is is not | 身份运算符 |
| in not in | 成员运算符 |
| and or not | 逻辑运算符 |

实例:

```
>>> a = 1
>>> b = 2
>>> c = 3
>>> e=a+b*c
>>> e
7
>>> e=(a+b)*c
>>> e
9
```
