# 1.3 解析库的安装

解析库:lxml、BeautifulSoup、PyQuery

## 1.相关链接

* PyPI:[https://pypi.org/](https://pypi.org/)
* whl:[https://www.lfd.uci.edu/~gohlke/pythonlibs/](https://www.lfd.uci.edu/~gohlke/pythonlibs/#lxml)

## 2.pip安装

```text
pip install 包名
```

## 3.wheel

```text
pip install xxx.whl
```

## 4.验证安装成功

```text
python3

>>> import 包名
```

如果没有报错，就代表安装成功了
