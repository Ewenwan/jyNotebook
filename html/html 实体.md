## HTML 实体

有些字符是不能使用的，比如说小于号和大于号，浏览器会误认为是标签

如果希望正确地显示预留字符，必须在HTML代码中使用字符实体

语法:

&entity\_name;

或者

&\#entity\_number;

小于号的表示实例:

```
&lt;
&#60
```

## HTML 中有用的字符实体
